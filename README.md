# Data Restore PDX

## Data Restore PDX - data recovery from damaged media - [https://datarestorepdx.vercel.com](https://https://datarestorepdx.vercel.com/?utm_source=github&utm_medium=readme)

### Installation & control

```sh
npm i
```

### Dev-mode

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Tech Stack

- React
- Next (App router) + SSR
- Tailwind CSS
- Radix-UI
- Lucide (Icons for React)
